如何让主页中的文章只显示摘要，而不是全部内容？

七牛云注册地址

在对 Hexo 进行配置前，需要说明一下，
网站根目录：~/my-blog
站点配置文件：~/my-blog/_config.yml
主题配置文件：~/my-blog/themes/next/_config.yml
因此，关于网站的配置会涉及到以上两个文件。

Front-matter

Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量。
可以通过修改模板目录 scaffolds 中的 post.md, draft.md 进行修改
修改 draft.md 创建草稿文件时添加到文件头部
修改 post.md 创建正式文件时添加到文件头部
https://hexo.io/zh-cn/docs/front-matter.html#分类和标签

Hexo 中分类(categories)和标签(tags)的区别

Hexo 配置网站显示 tags 和 categories 目录

  1. 要添加分类和标签，首先要添加tags和categories页面
  2. 主题的配置文件和站点的配置文件tags和categories的注释要打开

添加分类
hexo new page categories
INFO Created: ~/my-blog/source/categories/index.md
编辑 source/categories/index.md

确认站点配置文件里有category_dir: categories
确认主题配置文件里有categories: /categories

添加标签
hexo new page tags
INFO Created: ~/my-blog/source/tags/index.md
编辑 source/tags/index.md

确认站点配置文件里有tag_dir: tags
确认主题配置文件里有tags: /tags

在主题配置文件中，在menu项下，要把tags页打开
